## Что умеет приложение

**Главное окно (`MyForm`):**

- Показывает все направления в таблице (`DataGridView`):
  - номер направления;
  - название станции;
  - цена билета;
  - всего мест;
  - продано мест.
- Кнопка **«Обновить из БД»** – просто заново читает всё из таблицы `Routes`.
- **Добавить** – открывает второе окно для создания нового направления.
- **Изменить** – открывает второе окно с уже заполненными полями для редактирования.
- **Удалить** – удаляет выбранную строку из БД.
- **Сохранить в файл** – экспорт всех записей из БД в `.txt` файл (формат: `Number;Name;Price;CountPlace;SellTicket`).
- **Загрузить из файла** – читает `.txt` файл, очищает таблицу и загружает данные в БД.
- **Сорт. по имени** – сортирует направления по названию станции.
- **Сорт. по непроданным** – сортирует по сумме денег за непроданные билеты (по убыванию).
- **Стоимость непроданных билетов** – показывает окно с расчётом: сколько денег "лежит" в непроданных местах по каждому направлению.

---

## Второе окно – `EditForm`

Одно и то же окно работает в двух режимах:

- **Добавление** – вызывается как `EditForm()`:
  - можно вводить номер, название станции, цену, количество мест и проданных мест.
- **Редактирование** – вызывается как  
  `EditForm(number, name, price, countPlace, sellTicket)`:
  - поля заполняются текущими значениями,
  - номер направления менять нельзя.

Во всех случаях есть проверки:

- нельзя оставить пустое имя станции;
- номер > 0;
- цена > 0 и ≤ 10000;
- количество мест > 0;
- проданных мест от 0 до количества мест;
- при ошибке показывается `MessageBox` и запись не сохраняется.

После успешного `OK` данные **сразу пишутся в БД**, а главное окно обновляет таблицу.
